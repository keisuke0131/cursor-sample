# Cursorを使ったJava開発プロジェクト

## 概要

本プロジェクトは、Cursor AIアシスタントを活用したJava開発プロセスの説明とサンプルを提供します。
詳細設計、実装、単体テスト、結合テスト、システムテストの各フェーズにおける作業方法をドキュメント化しています。

## プロジェクト構成

```
プロジェクトルート/
├── docs/                          # 設計書・ドキュメント類
│   ├── 01_プロジェクト構造設計書.md
│   ├── 02_開発プロセスガイド.md
│   ├── 03_詳細設計書テンプレート.md
│   ├── 04_テスト設計書.md
│   ├── 05_サンプルシステム設計書.md
│   ├── 06_実装ガイドライン.md
│   └── 07_便利なコマンド集.md
│
├── src/                           # ソースコード（実装例）
├── test/                          # テストコード（実装例）
├── test-docs/                     # テスト設計書
└── README.md                      # 本ファイル
```

## 技術スタック

- **言語**: Java 17以上
- **フレームワーク**: Spring Boot 3.x
- **ビルドツール**: Maven
- **データベース**: H2 Database（開発環境）
- **テストフレームワーク**: JUnit 5
- **モックライブラリ**: Mockito

## 開発フロー

```
1. 詳細設計書作成
   ↓
2. 実装
   ↓
3. 単体テスト作成・実行
   ↓
4. 結合テスト作成・実行
   ↓
5. システムテスト作成・実行
   ↓
6. コードレビュー
   ↓
7. リリース
```

## Cursorの使い方

### 設計書の作成
```
@docs/03_詳細設計書テンプレート.md を参照して、
[機能名]の詳細設計書を作成してください
```

### 実装
```
@docs/05_サンプルシステム設計書.md の[クラス名]を実装してください。
パッケージは com.company.system.[レイヤー] です。
```

### テストの作成
```
@src/main/java/com/company/system/service/UserService.java の
単体テストを作成してください。
JUnit 5とMockitoを使用してください。
```

### コードレビュー
```
@src/main/java/com/company/system/service/UserService.java を
レビューしてください。改善点があれば指摘してください。
```

## プロジェクトの使い方

1. **設計書の確認**
   - `docs/`配下のドキュメントを確認
   - プロジェクト構造と開発プロセスを理解

2. **サンプルシステムの実装**
   - `docs/05_サンプルシステム設計書.md`を参照
   - Cursorを使用して実装

3. **テストの実装**
   - `docs/04_テスト設計書.md`を参照
   - 単体テスト、結合テスト、システムテストを実装

4. **実装ガイドラインの確認**
   - `docs/06_実装ガイドライン.md`を参照
   - コーディング規約に従って実装
