# リファクタリング

指定されたコードをリファクタリングしてください。

## リファクタリング方針
- @.cursor/rules/coding-standards.md のコーディング規約に従う
- コードの可読性を向上させる
- 重複コードを排除する
- メソッドの責任を明確にする
- 適切な命名規則を使用する
- パフォーマンスを考慮する

## リファクタリング項目
1. メソッドの抽出
2. 変数名の改善
3. マジックナンバーの定数化
4. 条件分岐の簡素化
5. コメントの追加・改善
6. 不要なコードの削除

## 注意事項
- 既存の機能を壊さないこと
- テストが通ることを確認すること
- 変更内容を明確に説明すること
