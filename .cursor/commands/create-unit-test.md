# 単体テスト作成

指定されたクラスの単体テストを作成してください。

## 実装方針
- @.cursor/rules/testing.md の単体テストガイドラインに従う
- JUnit 5とMockitoを使用
- テストクラス名は `[対象クラス名]Test` とする
- パッケージ構造は `src/test/java/com/company/system/[レイヤー]/` とする
- テストメソッド名は `test[メソッド名]_[シナリオ]` 形式とする

## テストケース
- 正常系のテストケース
- 異常系のテストケース（例外処理）
- 境界値テスト
- モックの適切な使用

## 対象ファイル指定
現在開いているファイル、または `@ファイルパス` で指定されたファイルの単体テストを作成してください。
